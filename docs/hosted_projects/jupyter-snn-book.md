### Proposed Repository Name

jupyter-snn-book

### Short Description

A fork of the [jupyter book](https://github.com/jupyter-book/jupyter-book) repository with added support for dynamical systems visualization

### Is this a fork of an existing project or a brand-new repository?

Yes, this forks [jupyter book](https://github.com/jupyter-book/jupyter-book)

### Why should this project be hosted in the Open Neuromorphic organization?

This ties to the SNN book project and permits the visualization of dynamical systems. This is relevant to ONM because it'll be used for the SNN book and because it's an important part of teaching neural/spiking systems.

### Are there existing open-source projects that do this? If so, how does this differ?

None
